REsearch program made by NS174 - Natch Sadindum - 1269188
This program takes a FSM file called "machines.txt" and a text file
* Usage is - java REsearch <textfile>
* The FSMs in the text file should be in the format "<stateName>,<character>,<nextState1>,<nextState2>" and each FSM should be on its own line
* eg. 0,a,0,1 - 5,@,2,5 and s
* the first state is considered the start state, and the last state the final state
